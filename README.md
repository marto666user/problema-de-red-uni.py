# Optimizador Red Universidad

Resuelve programación lineal para conectar 19 salones con mínimo costo usando PuLP.

## Instalación
pip install -r requirements.txt

## Usopython network_optimizer.p
y
## Parámetros
Modificá los valores en la función resolver_red():
- puertos_necesarios: 494 puertos mínimo
- ancho_necesario: 3800 Mbps mínimo 
 - energia_max: 700 Watts máximo
